# ⚽ Football League Management System (EF Core)

## 📌 Overview
This project is a console-based backend application developed using **C# and Entity Framework Core (EF Core)**.  
It simulates a football league management system where teams, players, stadiums, and matches are managed, and league statistics are dynamically updated.

---

## 🚀 Features
- Full CRUD operations for:
  - Teams
  - Players
  - Stadiums
  - Matches
- Dynamic league table calculation (points, wins, losses, goals, etc.)
- Player performance tracking (goals scored)
- Automatic update of team and player statistics after each match
- Data validation (unique constraints, relationships)

---

## 🧠 Technical Implementation
- Built using **C# (.NET Console Application)**
- Database operations handled with **Entity Framework Core**
- Designed relational data models and entity relationships
- Implemented:
  - **N-tier architecture**
  - **Repository pattern**
  - **Service layer**
- Applied **clean code principles**
